# Onboarding

> Embrace what you don’t know, especially in the beginning, because what you don’t know can become your greatest asset.
>
> —[Sara Blakely]

Onboarding must be:

1. Repeatable
1. Self Service
1. Written Down
1. In Source Control

Additionally, _every new hire must update the docs_.
